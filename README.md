# FDT - Fraud Detection in UPI

## Project info

**FDT** (Fraud Detection in UPI) is a real-time fraud detection system for UPI transactions using ensemble machine learning models.

## How to run locally

```sh
# Navigate to the project directory
cd lavender-shield

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Technologies used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Deploy to any static hosting service (Vercel, Netlify, Render, etc.)
